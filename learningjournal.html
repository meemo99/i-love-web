  • 29-09-2025

    • Wat is het verschil tussen flexbox en grid? Wanneer zou je welke gebruiken?
      • Bij grid maak je gebruik van een grid. Je kan zelf bepalen hoeveel rows en columns je gebruikt en waar je je divs plaatst. Bij flex-box gebruik je eigenlijk maar 1 row en 1 column.
      • Flex is handig voor het centreren en alignen van elementen.
    • Wat is de default layout mode van elk element?
      • Block-layout is de default.
    • Welke layout heb je per onderdeel nodig voor het ontwerp van de opdrachtgever?
      • Ik denk dat ik vooral gebruik ga maken van Flexbox en waar nodig grid.

  • 01-10-2025

    • 3 eigenschappen van leesbare text.

      De drie eigenschappen van leesbare tekst zijn:
    • Minimale font-size van 16px.
    • Maximale breedte van 10-12 woorden.
    • Een line height van 1.4.

    Welke stappen moet je nemen om Mobile First te gaan werken en hoe je dit aan je code kunt zien.
    • Mobile First is te zien aan de mediaqueries in CSS. En bij mediaqueries wordt gebruik gemaakt van de (min-width) voor de breakpoints.
