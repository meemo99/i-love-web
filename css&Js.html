JS Fundamentals

Basic Functions

Below is a function called 'sum'. This means you can call any number and it will return the sum( + ). sum(1,3) will return 4, sum(2,5) will return 5etc.

Example;

                    function sum(x,y){
                    return x + y}
There must be a return from inside the function, otherwise the function will return undefined.

Strings

To create a string in JavaScript use double quotes(") or single quotes(').


Example;

                    "This is a string";
                    'This is another string!'

String property

.length property is used to return the length of the string.

Example;

                    "Nice!".length;
                    //5
Now you have the length of the string ""Nice!"
To call on a variable you simply put the name of the variable before the .length
Example;

                let text = "Hello World";
                text.length; // 11

CSS Challenges

View Transition

View Transitions maakt "naadloze, visuele overgangen tussn verschillende weergaven op uw website. De overgangen wordem aangestuurd door CSS-animaties Een overgang in dezelfde document weergave wordt aangeroepen met een if..else statement: 

if (document.startViewTransition) {} else {}
Een overgang tussen document(cross-document) weergaven wordt in CSS toegepast met: 
@view-transition {navigation:auto;}
De duration van de view-Transition kan je veranderen met: 
::view-transition-group()
Tussen de haakjes kan je b.v de 
(*)
selector kiezen of b.v 
(:root)

Door view-transition-name te gebruiken kan je elementen specifiek aanroepen voor de View Transitions. 
h1{view-tranitions-name: howdy}
Vervolgens gebruik je ::view-transition_group(howdy){animation-duration: } om zo de duration aan te passen alleen op het gekozen element.
